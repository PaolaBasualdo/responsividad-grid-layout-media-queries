Sistema de Gestion de Stock

Este sistema de gestión de stock para una ferretería, desarrollado en Python, permite administrar productos y categorías con funcionalidades CRUD (Crear, Leer, Actualizar, Eliminar). Los usuarios deben loguearse con distintos tipos de permisos (administrador, empleado), lo que define las acciones disponibles en el menú principal. Los productos se organizan por categorías almacenadas en archivos de texto plano (Categorias.txt), y la información del inventario se guarda en un archivo JSON (productos.json) para garantizar la persistencia de los datos. Además, el sistema permite generar reportes sobre el estado del stock y productos de interés. La aplicación está estructurada en módulos (logeo_usuario.py, menu_productos.py, sistema_gestion_stock.py) que separan la lógica de autenticación, gestión de productos y reportes, facilitando la escalabilidad y el mantenimiento del código. Es una solución sencilla, ideal para ferreterías pequeñas, que no requiere bases de datos complejas.
